## Assignment 1

The first assignment focuses on performing Exploratory Data Analysis (EDA) on multiple machine learning datasets.

### Objectives

- Import and preprocess datasets
- Understand dataset characteristics
- Perform statistical analysis
- Visualize feature distributions
- Identify missing values and outliers
- Generate useful insights before model building

---

## Datasets Included

### 1. Iris Dataset
- Multi-class flower classification
- Commonly used for classification algorithms

### 2. Diabetes Dataset
- Patient health records
- Used for diabetes prediction

### 3. Email Classification Dataset
- Email spam/ham classification
- Useful for Natural Language Processing (NLP)

### 4. Handwritten Letter Dataset
- Images of handwritten English characters
- Suitable for Computer Vision tasks

### 5. Loan Dataset
- Loan approval prediction
- Binary classification problem
